Selected work

Ziva Health — EHR Integration Bridge

Backend owner · end-to-end

Multi-tenant Django microservice that bridges Ziva Health with 20+ EHRs (Healthie, Elation, and others). Syncs patient records, SOAP notes, clinical documents, and appointments over REST & GraphQL, with webhook-driven real-time exchange. Configurable per-clinic templates let new EHRs onboard via config rather than code — and a single request/response contract across all adapters keeps integration bugs rare.

  • Django
  • REST
  • GraphQL
  • Webhooks
  • Multi-tenant

Isabella AI — HIPAA-Compliant Healthcare AI

Backend engineer

FastAPI microservice powering natural-language conversations between patients and providers, built on LangGraph + LangChain. Stateful conversation graphs keep clinical context across sessions; structured patient history (past notes, appointments, records) grounds every LLM response. HIPAA-compliant by design — PHI isolation, audit logging, and secure handling baked in.
